## How to Upload Audio Files

1. **Via GitHub Web Interface:**
   - Go to https://github.com/bilalod1-collab/storage/tree/main/audio
   - Click "Add file" > "Upload files"
   - Drag and drop your .opus or .ogg files
   - Commit the changes

2. **Via Git Commands:**
   ```bash
   git clone https://github.com/bilalod1-collab/storage.git
   cd storage/audio
   # Copy your audio files here
   git add .
   git commit -m "Add audio files"
   git push
   ```

## n8n Access URLs

Once you upload audio files, they'll be accessible at:
```
https://raw.githubusercontent.com/bilalod1-collab/storage/main/audio/YOUR_FILENAME.ogg
```
